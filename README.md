# Vue 3 + TypeScript + Vite + PixiJS

A simple boilerplate that loads PixiJS into a Vite + Vue instance.

## Basic Setup

```ts
npm run install

npm run dev
```

## What is Included

* Vue Hooked in with Pixi.js
* Basic Player Movement (WASD)
* Basic Player Sprite (It's a Ball)
* A Simple Scene Loader
